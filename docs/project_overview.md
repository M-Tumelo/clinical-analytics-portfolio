# Project Overview: Clinical Data Analytics & Visualization

## Objective
This project demonstrates an end-to-end clinical data workflow, including:
- Exploration and validation of SDTM datasets
- Creation of analysis-ready datasets (ADaM-like)
- Development of an R Shiny dashboard for clinical data review and safety monitoring

## Scope
The project focuses on three key SDTM domains:
- DM (Demographics)
- AE (Adverse Events)
- LB (Laboratory Data)

## Workflow
1. Load and explore SDTM datasets
2. Perform data validation and quality checks
3. Transform data into analysis-ready structures
4. Develop interactive visualizations using R Shiny

## Tools & Technologies
- R (tidyverse)
- R Shiny
- Git (version control)

## Outcome
- Reproducible data processing scripts
- Structured datasets for analysis
- Interactive dashboard supporting data review and safety insights
